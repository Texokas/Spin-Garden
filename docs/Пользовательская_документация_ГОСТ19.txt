1. Назначение программы

Данный программный продукт представляет собой Telegram-бота для игры в казино. Бот позволяет пользователям играть в популярные азартные игры (21, рулетка, крутилка), отслеживать баланс, просматривать таблицу лидеров и получать справку по командам.

2. Возможности программы
- Игра в "21"
- Игра в рулетку (ставки на цвет, чет/нечет, зеро)
- Игра "Крутилка"
- Просмотр баланса
- Просмотр таблицы лидеров
- Получение справки по командам

3. Требования к вводу и выводу
- Ввод осуществляется через команды Telegram и нажатия на кнопки меню.
- Вывод — текстовые сообщения, кнопки, игровые результаты, уведомления об ошибках.

4. Описание работы
1. Пользователь запускает бота командой /start.
2. В главном меню выбирает игру или действие (баланс, лидеры, помощь).
3. В каждой игре пользователь делает ставку, получает результат, может сыграть снова или вернуться в меню.
4. Баланс автоматически обновляется после каждой игры.
5. Для получения справки используется команда /help или кнопка "Помощь".

5. Сообщения об ошибках
- "Недостаточно монет. Минимальная ставка: ..."
- "Пользователь не найден. Используйте /start"
- "Произошла ошибка. Пожалуйста, попробуйте позже."
- "Неверный тип ставки"

6. Примеры работы
- Запуск: /start
- Проверка баланса: /balance
- Получение справки: /help
- Игра в рулетку: выбор типа ставки, получение результата
- Игра в 21: последовательные действия (взять карту, стоп, удвоить)
- Крутилка: запуск, результат, повтор

7. Ограничения
- Для игры требуется Telegram-аккаунт
- Минимальный баланс для ставок
- Административные функции доступны только определённым пользователям
